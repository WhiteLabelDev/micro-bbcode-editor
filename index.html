Micro BBCode Editor

A very small, simple, and easily extensible real-time editor supporting basic BBCode tags. It supports applying tags to any text selection or adding them at the caret if not. You can use keyboard shortcuts (hover your mouse over toolbar to see what they are). It also has a liberal parser which ignores any unknown tags, and shows help where any problems are if you manage to type invalid markup. Licensed under MPL2 - see GitHub home.
